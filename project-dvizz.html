DIALOG VIZZ

HTML5, CSS3, JQUERY, JAVASCRIPT, PHP, MYSQL

Dialog Vizz Mobile is a dedicated Pre-Paid Mobile SIM for the Sri Lankan and Asian market launched by QiComm Limited providing customers.

I had the task of building a standard web site for Dialog Vizz Mobile with clean code. I worked with the in-house designers and built all of the front-end from scratch.

As a result I developed fully interactive and highly-visual HTML5, CSS3 and jQuery website. To show user information I used PHP with MySQL.

Once Search engine optimization was implemented it competes extremely well in search ranking in comparison to well established competitors in the MVNO industry.

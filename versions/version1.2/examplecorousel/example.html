jQuery Carousel Evolution

The Web's Most Powerful Carousel Slider

Features

Configurable Image

Allows you to set the size of the front and back image. configurable number of images visible per scroll. Also set the position of the image.

9 Different Styles

You can create up to 9 different styles of carousel with this plugin.

Image Reflection

Displays reflection effects to your images. No more photoshop. The reflections appear instantly under each image. You are also allowed to specify the height of the reflection.

Shadow

Automatically create shadow effect for each image.

Text Description

You can add text description that associated with each image, and placing it anywhere in the web page. See example preview.

Autoplay

Automatically animate through your carousel when visitors arrive on your website. Great for header sliders. You can set interval time of autoplay.

Navigation Button

Navigate through carousel image with next/prev button, or bullets and numbers button.

Video Support

jQuery Carousel Evolution can also play video from YouTube and Vimeo.

Cross-browser Support

Displays consistently across all major browsers including Internet Explorer, Firefox, Opera, Chrome, and Safari.

Multiple Carousels

You can setup multiple carousels on a single web page and skin each one differently.

Unlimited Images

Support unlimited number of images.

Mouse Control

Enhance the experience of your carousel by allowing users to seamlessly scroll through slides.

Callback Functions

Run functions before and after animating between slides on your carousel.

Public API

The carousel plugin offers a public API which you can use to control component inside the slider from within your own scripts.

Search Engine Ready

All content is easily accessible to search engines and markup semantically correct to help you achieve the best possible SEO results.
